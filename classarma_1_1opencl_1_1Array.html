Detailed Description

template<class T, int N>
class arma::opencl::Array< T, N >

A wrapper for blitz::Array class for OpenCL kernels.

Array optimises data transfer between host and device by explicitly copying data to accelerator, or allocating memory directly on the device. These operations are tied to the object rather than OpenCL kernel, so there is no need to copy data before/after each kernel execution. In all other cases this class acts as blitz::Array.
